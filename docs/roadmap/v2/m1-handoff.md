# V2-M1: Handoff

> spawn a new thread from summarized prior context

## Goal

Support continuation of long-running work through a fresh thread seeded by structured summary rather than the full original transcript.

## Scope

- handoff summary format
- new-thread spawn flow from prior work
- context carry-over rules
- resume UX for follow-up execution

## Dependencies

- compact summaries preserve enough task state to resume work safely
