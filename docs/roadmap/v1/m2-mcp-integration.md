# V1-M2: MCP Integration

> stdio + Streamable HTTP + tool discovery + dynamic registration

## Goal

Open Scorel to external tool ecosystems by making MCP a first-class tool source alongside built-in tools.

## Scope

- stdio transport support
- Streamable HTTP transport support
- server capability discovery
- dynamic tool registration and lifecycle handling

## Dependencies

- runner/tool registry abstractions ready for extension
- permission model able to represent externally sourced tools
