# V1-M1: Autonomy Foundation

> auto_compact + subagent + TodoWrite + permission system

## Goal

Give Scorel the baseline autonomy required for longer and more complex tasks without depending on constant manual steering.

## Scope

- auto_compact triggered by context thresholds
- subagent execution with isolated child context and summary-only return path
- TodoWrite-style planning and progress tracking
- permission system with global and per-tool control

## Dependencies

- V0 compact and session model stable
- Tool execution and approval flow mature enough to extend
